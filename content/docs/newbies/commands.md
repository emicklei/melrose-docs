## Help

Show all available commands and functions.

```bash
:h
```

### MIDI connectivity

Show all availale input and output devices.

```bash
:m
```

### Immediately stop all sounds

Abort playing of any music object.

```bash
:k
```

### Exit program

Stop the melrose program and all playing sounds.

```bash
:q
```
