Musical objects

Create a Note, Sequence, Chord, Progression to start your musical journey.

Operations

Apply to musical objects such Repeat, Requence, Replace, Pitch, Fraction, Mapping to create new musical combinations.

Playing

Play any musical object directly, or in a Loop or in a Track, alone or together with others.

Interactive

While playing you can change any musical object by re-evaluating object definitions.

MIDI

Send MIDI to any sound producting system such as a DAW or your hardware synthesizer. React or record MIDI input to create new musical objects.
